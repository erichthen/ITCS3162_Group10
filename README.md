# California Housing Market Intelligence

## ITCS 3162: Data Mining Course Project -- Group #10

### By Erich Then, Eric Ho, Aaron Putnum, and Ethan Park

---

### Objective:
Working with an older, real-world geospatial dataset to predict median house values across California. The project follows the complete data science pipeline, including team coordination, critical data investigation, regression modeling, error mitigation, and a final professional presentation. Click [here](/documents/project_overview.pdf) to see the full overview/description for this project. (It is in /documents).

---

### Key Requirements:  
- A fully automated KNIME pipeline that runs from raw [housing.csv](/data/housing.csv) to final predictions with manual intervention.  

- Weekly group meetings and completion of the audit form. 

- Economic Justification: Final report must identify which variables are redundant based on linear correlation values rather than intuition.
